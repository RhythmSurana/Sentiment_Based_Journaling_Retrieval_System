# 🌟 Lumina Journal

**Lumina Journal** is an AI-integrated platform that turns your daily reflections into actionable insights. Stop just recording your life—start understanding it.

## ✨ Key Features
- **🤖 AI Reflection**: Automatically scores your mood (1-10) and summarizes every entry using Gemini 3 Flash.
- **📈 Mood Dashboard**: Visualizes your emotional journey with interactive charts.
- **💬 Lumina Assistant**: A personal chatbot that "reads" your history to help you find patterns and insights.
- **🔐 Privacy First**: Secure Google Login and private database encryption via Firebase.

## 🛠️ Tech Stack
- **Frontend**: React 19, TypeScript, Tailwind CSS
- **AI Engine**: Google Gemini 3 Flash API
- **Backend**: Firebase (Auth & Firestore)
- **Charts**: Recharts
